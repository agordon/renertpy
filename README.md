the [docs](docs/) directory.


Installation
============

From PyPi (latest released version):

    sudo pip install renertpy

From source code:

    git clone https://github.com/agordon/renertpy.git
    cd renertpy
    python setup.py build
    python setup.py test
    sudo python setup.py install
