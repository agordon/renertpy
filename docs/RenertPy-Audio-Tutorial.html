Play Frequencies - play_frequencies( list [,duration-in-seconds] )¶

Accepts frequency (HZ) values from 200 to 11000 . \ Each frequency is played for the same duration (default = 0.3 seconds). \ Example: The common tuning reference note of Middle-C "A" is 440hz. \ See Table of Musical Notes and Their Frequencies for more details.
